# Nexus Architecture Decisions

> State: Index
> Governance: This directory stores compact architecture decision records.

Use this directory for decisions that change documentation lifecycle, runtime ownership, architecture boundaries, or long-term product direction. Keep ADRs short: context, decision, consequences, and status.

## Decisions

| Decision | Status | Scope |
| --- | --- | --- |

## 中文概述

### 作用

`decisions/` 存放短 ADR，记录关键架构和治理选择。

### 边界

ADR 不替代 TODO、WORK_LOG 或具体实现文档；它只记录为什么选择某条长期规则。
