# BabeL-O v0.2.6 Release Notes

Release date: 2026-05-31

## Summary

v0.2.6 continues the Nexus-first direction and prepares BabeL-O for a cleaner standalone binary distribution path. This release also updates the public project positioning from a coding-only assistant toward a generalized agent runtime while keeping the lightweight `bbl` CLI as the primary interaction surface.

## Highlights

- Updated README positioning for BabeL-O as a Nexus-first generalized AI agent.
- Added source-build and standalone native binary build documentation.
- Added `npm run build`, `npm run build:binary`, and `prepack` scripts.
- Added a Node.js SEA build script with embedded built-in skill assets.
- Added a production-aware `bin/bbl.js` launcher that uses `dist/cli/program.js` when available and falls back to the TypeScript development entry in source mode.
- Added a hidden `__server` CLI entry for starting Nexus from compiled output.
- Added native SEA asset loading for built-in skills, with filesystem loading preserved for source development.
- Updated runtime TODO documentation around context management accuracy, compact hardening, working set, prefix cache, and path mention follow-ups.

## Build And Packaging

- New build config: `tsconfig.build.json`.
- New build script: `scripts/build-binary.js`.
- New development dependencies: `esbuild` and `postject`.
- `.cache/` is ignored because the SEA build can download and cache official Node.js runtime binaries locally.

## Notes

- The release notes live in `docs/releases/` as project-level version records.
- Nexus implementation plans and runtime TODOs remain in `docs/nexus/`.
